- React
- Apollo Client
- React Router Dom
- React Hook Form
- yup
- React Table
- styled-components

**Functionalities**

- Create a user
- Create a team

- See teams
- Update a user

**UI**

- Homemade UI without framework

## Running the project

Clone this repository :

```
git clone https://github.com/GuillaumeBe/User-Management-Front-End.git
cd User-Management-Front-End
```

Install packages :

```
npm install
```

When installation is complete, run the project with:

```
npm start

```
